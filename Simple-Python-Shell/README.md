# 💻 Simple Python Shell (CLI)

## 📌 Overview

**English:**
Simple Python Shell is a lightweight command-line interface that allows users to execute system commands directly from a Python script. It acts as a minimal wrapper around the system shell.

**العربية:**
مشروع **Simple Python Shell** هو واجهة سطر أوامر بسيطة مكتوبة بلغة بايثون، تتيح تنفيذ أوامر النظام مباشرة من داخل البرنامج.

---

## ⚙️ Features | الميزات

* 🖥 Execute system commands
* 🔁 Continuous command loop
* 🎨 Colored prompt using `termcolor`
* 🚪 Exit using `exit` or `stop`

---

## 🧪 Usage | طريقة الاستخدام

### English:

1. Run the script
2. Enter any system command
3. Output will be displayed directly
4. Type `exit` or `stop` to quit

### العربية:

1. شغّل البرنامج
2. أدخل أي أمر للنظام
3. سيتم عرض النتيجة مباشرة
4. اكتب `exit` أو `stop` للخروج

---

## 🚀 Installation & Run | التشغيل

### 1. Install dependency:

```bash
pip install termcolor
```

### 2. Run:

```bash
python main.py
```
